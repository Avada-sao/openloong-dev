介绍

本仓库为基于mujoco的针对OGHR人形机器人的控制仿真测试。mujoco的仿真引擎、pinocchio动力学库、eigen、quill记录工具、GLFW图形库、jsoncpp解析库等也包含到了仓库之中，但仿真界面需系统支持openGL，通过安装 sudo apt install libglu1-mesa-dev freeglut3-dev

控制测试工程为OGHR_project中的相关内容。运行该程序的步骤为：

  1. 在OGHR_project的文件夹中打开ubuntu终端，输入以下指令
  2. mkdir build
  3. cd build
  4. cmake ..
  5. make
  6. ./walk_mpc_wbc 或 ./walk_wbc

然后即可看到mujoco仿真界面。
