本仓库为基于mujoco的针对\+OGHR人形机器人的控制仿真测试。mujoco的仿真引擎、pinocchio动力学库、eigen、quill记录工具、\+GLFW图形库、jsoncpp解析库等也包含到了仓库之中，但仿真界面需系统支持open\+GL，通过安装 {\ttfamily sudo apt install libglu1-\/mesa-\/dev freeglut3-\/dev}

控制测试工程为\+OGHR\+\_\+project中的相关内容。运行该程序的步骤为：


\begin{DoxyEnumerate}
\item 在\+OGHR\+\_\+project的文件夹中打开ubuntu终端，输入以下指令
\item mkdir build
\item cd build
\item cmake ..
\item make
\item ./walk\+\_\+mpc\+\_\+wbc 或 ./walk\+\_\+wbc
\end{DoxyEnumerate}

然后即可看到mujoco仿真界面。
